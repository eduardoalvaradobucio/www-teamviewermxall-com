geomaroc: Easily Visualize Geographic Data of Morocco

Tools to easily visualize geographic data of Morocco. This package interacts with data available through the 'geomarocdata' package, which is available in a 'drat' repository. The size of the 'geomarocdata' package is approximately 12 MB.
