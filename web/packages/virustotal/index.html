virustotal: R Client for the 'VirusTotal' API

Provides a comprehensive R interface to the 'VirusTotal' API v3.0 <https://docs.virustotal.com/>, a Google service that analyzes files and URLs for viruses, worms, trojans and other malware. Features include file/URL scanning, domain categorization, passive DNS information, IP reputation analysis, IoC relationships, sandbox analysis, and comment/voting systems. Implements rate limiting, error handling, and response validation for robust security analysis workflows.
