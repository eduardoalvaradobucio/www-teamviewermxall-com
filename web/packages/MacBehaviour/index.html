MacBehaviour: Behavioural Studies of Large Language Models

Efficient way to design and conduct psychological experiments for testing the performance of large language models. It simplifies the process of setting up experiments and data collection via language models’ API, facilitating a smooth workflow for researchers in the field of machine behaviour.
