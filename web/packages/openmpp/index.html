openmpp: Programmatic Interface to 'OpenM++'

A programmatic interface to the 'OpenM++' microsimulation platform (<https://openmpp.org>). The primary goal of this package is to wrap the 'OpenM++' Web Service (OMS) to provide 'OpenM++' users a programmatic interface for the R language.
