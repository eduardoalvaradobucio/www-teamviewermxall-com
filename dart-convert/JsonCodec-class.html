JsonCodec
description

JsonCodec class final

A JsonCodec encodes JSON objects to strings and decodes strings to JSON objects.

Examples:

var encoded = json.encode([1, 2, { "a": null }]);
var decoded = json.decode('["foo", { "bar": 499 }]');
