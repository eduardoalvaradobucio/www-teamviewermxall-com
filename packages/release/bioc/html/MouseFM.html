MouseFM

This is the released version of MouseFM; for the devel version, see MouseFM.

In-silico methods for genetic finemapping in inbred mice

DOI: 10.18129/B9.bioc.MouseFM


Bioconductor version: Release (3.22)

This package provides methods for genetic finemapping in inbred mice by taking advantage of their very high homozygosity rate (>95%).

Author: Matthias Munz [aut, cre] ORCID: 0000-0002-4728-3357 , Inken Wohlers [aut] ORCID: 0000-0003-4004-0464 , Hauke Busch [aut] ORCID: 0000-0003-4763-4521

Maintainer: Matthias Munz <matthias.munz at gmx.de>

Installation

To install this package, start R (version "4.5") and enter: 


if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install("MouseFM")

For older versions of R, please refer to the appropriate Bioconductor release.
