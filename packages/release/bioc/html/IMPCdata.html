IMPCdata

This is the released version of IMPCdata; for the devel version, see IMPCdata.

Retrieves data from IMPC database

DOI: 10.18129/B9.bioc.IMPCdata


Bioconductor version: Release (3.22)

Package contains methods for data retrieval from IMPC Database.

Author: Natalja Kurbatova, Jeremy Mason

Maintainer: Jeremy Mason <jmason at ebi.ac.uk>

Citation (from within R, enter citation("IMPCdata")):

Installation

To install this package, start R (version "4.5") and enter: 


if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install("IMPCdata")

For older versions of R, please refer to the appropriate Bioconductor release.
